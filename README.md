# qrcode-rust-wasm-bindgen

[wasm-bindgen](https://github.com/rustwasm/wasm-bindgen) build of [qrcode-rust](https://github.com/kennytm/qrcode-rust)

## example

```sh
npx @hiogawa/qrcode-rust-wasm-bindgen https://github.com
```
